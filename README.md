# Awesome Game Design 

A curated collection of game design resources for designers, developers, students, and anyone interested in creating better games.

The goal of this repository is to gather high-quality resources in a single place, including books, articles, talks, tools, research papers, templates, communities, and more.

## Categories

- 📚 Books
- 🎥 Talks & Conferences
- 📝 Articles & Blogs
- 🛠 Design Tools
- 📄 Templates
- 📊 Game Economy & Balance
- 🧠 Psychology & Player Experience
- 🎮 Level Design
- 🎲 Systems Design
- 📖 Narrative Design
- 🎨 UX/UI
- 📑 Academic Papers
- 🌍 Communities
- 🤖 AI for Game Design
